- Next.js에서는 SSG와 SSR을 위한 데이터 패칭 기능을 제공합니다.
- 사이트를 렌더링하기 전 어떤 데이터를 이용해서 페이지를 생성할 것인지에 대한 함수를 작성하는 기능을 말하는 것이죠

---

getStaticProps는 getServerSideProps와 비슷하지만 Static은 정적 사이트 생성을 위해 사용됩니다. 빌드 시 데이터를 패치합니다

```javascript
function BlogPost({ post }) {
  return (
    <div>
      <h1> {post.title} </h1>
      <div> {post.content} </div>
    </div>
  );
}

export async function getStaticProps() {
  const res = await fetch("https://.../posts/1");
  const post = await res.json();
  return {
    props: {
      post,
    },
  };
}

export default BlogPost;
```
