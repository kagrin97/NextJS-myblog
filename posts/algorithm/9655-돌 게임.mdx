<h2 className={`dark:text-white`}>문제 설명</h2>

dp문제답게 하나씩 해보면 점화식이 보인다

i % 2 === 0 => 창영이가 이김
else => 상근이가 이김

---

<h2 className={`dark:text-white`}>코드</h2>

```js showLineNumbers
let input = require("fs")
  .readFileSync(process.platform === "linux" ? "dev/stdin" : "input.txt")
  .toString()
  .trim();

input = +input;

if (input % 2 === 0) {
  console.log("CY");
} else {
  console.log("SK");
}
```
